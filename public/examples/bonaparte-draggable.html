About

A draggable, sortable list component. A 'draggable.drop' event is fired on the tag when an item is dropped. this ievent contains a details object with dropedElem - the current dragged element, dropZone - the element that the dragged element was dropped on to and order - the nodes in the updated order.

Attributes

Handler

Optional - [querySelector] - Changes the drag handler from the children to selector within a child

Default .buttons

Range

Optional - [start_index, end_index] - sets the class of inRange of the indexed children

Default 0,2

Target (coming soon)

Optional - [querySelector] - NOT IN USE

Update DOM

Optional - [boolean] - Default true, used when using frameworks like React that get upset if you update the DOM

Default false
